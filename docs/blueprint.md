# **App Name**: ResolveX

## Core Features:

- Secure Registration: User registration via email verification using SendGrid for OTP delivery and Firebase Auth for authentication.
- Grievance Submission: Users can submit grievances by taking a photo, which automatically captures the GPS location and stores the data in Firestore.
- Live Grievance Tracking: Real-time updates of grievance locations on a Google Maps interface centered around KLH University Hyderabad, using red pins to indicate issues.
- User Dashboard: A personalized dashboard for users to view their submitted complaints and their current status.

## Style Guidelines:

- Primary color: Vibrant purple (#9D4EDD), reflecting the modern technology.
- Background color: Dark gray (#1E1E1E), creating a sleek dark mode interface as requested by the user.
- Accent color: Electric blue (#48BFE3) for neon buttons and interactive elements, to highlight key actions and enhance the user experience.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral look.
- Use simple, geometric icons to represent different types of grievances, ensuring clarity and ease of understanding.
- Implement a clean and intuitive layout with a focus on map integration. Ensure ease of navigation, prioritizing quick access to grievance submission and tracking features.
- Subtle animations for pin updates and status changes, providing visual feedback to the user and enhancing the dynamic feel of the application.
